int(a[i]) # 使用时要转换为整数

n = int(input())
numlist = []

用 map 输入

第一行四个整数 a,b,c,ma,b,c,ma,b,c,m, 第二行 a∗b∗ca*b*ca∗b∗c 个整数

A,B,C,m = map(int, input().spilt())
life = list(map(int, input().spilt()))

第一行两个整数 nnn 和 kkk, 后面 nnn 行，每行两个整数 h,wh,wh,w

n,k = map(int, input().spilt())
w = []

for i in range(n):
